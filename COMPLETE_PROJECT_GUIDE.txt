COMPLETE PROJECT GUIDE - FINAL REPO VERSION

Project name:
ResNet34-Based Galaxy Morphology Classification with Machine Unlearning

1. Final purpose
This project combines galaxy morphology classification and machine unlearning in one pipeline. The classifier learns galaxy morphology from Galaxy Zoo 2 images. Machine unlearning is then used to reduce the influence of intentionally mislabeled training samples.

2. Final dataset
- Dataset: Galaxy Zoo 2
- Matched images used by loader: 61,578
- Final classes:
  - Smooth
  - Featured/Disk
  - Artifact
- Current mapped class counts in the repo dataset:
  - Smooth: 26,693
  - Featured/Disk: 34,826
  - Artifact: 59

3. Final model
- Backbone: ResNet34
- Pretrained on ImageNet: yes
- Input resolution: 224 x 224
- Split: 70% train / 15% validation / 15% test
- Seed: 42

4. Final training code
The main training code is train.py.
The current code uses:
- AdamW optimizer
- Focal loss by default
- mixed precision on CUDA
- cosine annealing scheduler
- early stopping
- best checkpoint reload before final evaluation/unlearning

5. Final saved training result
From training_log.csv:
- Best validation accuracy: 85.58%
- Best balanced accuracy: 87.56%
- Best macro-F1: 76.41%
- Epochs completed in saved run: 19

6. Final unlearning methods
- Gradient Ascent
- Fisher Forgetting
- Full Retrain

The standalone unlearning script is unlearn.py.

7. Final saved unlearning result
From unlearning_results.csv:
- Gradient Ascent: test 70.80%, forget 11.01%, retain 76.39%, time 103.57 s
- Fisher Forgetting: test 41.71%, forget 28.36%, retain 43.44%, time 213.76 s
- Full Retrain: test 76.17%, forget 8.36%, retain 99.64%, time 2649.43 s
