Project title:
ResNet34-Based Galaxy Morphology Classification with Machine Unlearning

What this repo contains:
- Final ResNet34 classifier code
- Machine unlearning implementation
- Saved checkpoints
- Saved training and unlearning results
- Streamlit app for demo and presentation

Best first files to read:
1. README.md
2. FINAL_PROJECT_FACTS.md
3. HOW_TO_RUN.md
4. app.py

Quickest way to demo the project:
1. Make sure dependencies are installed
2. Run: streamlit run app.py
3. Use the saved model and saved results already present in the folder

Important:
- You do not need to retrain every time to run the app.
- The final project is based on ResNet34, not ResNet18.
- The most trustworthy saved result files are training_log.csv and unlearning_results.csv.
